# 2025 클라우드 웹 서비스 과제 요구사항 정리

## 📋 과제 개요

### 1. 설계 과제 (11월 20일 제출)
**과제명**: 공공데이터포털 활용 스마트시티 어플리케이션 설계

**요구사항**:
1. ✅ **RESTful API 설계** (필수, 미제공 시 감점)
2. ✅ **스마트폰 앱과 PC 앱 모두 지원**
3. ✅ **서버리스 및 마이크로서비스 아키텍처** (Amazon Lambda 사용 필수, 미사용 시 감점)
4. ✅ **DBMS 사용** (필수, 미사용 시 감점)
   - Amazon RDS 사용 가능
   - **MongoDB 사용 시 가산점** ⭐

### 2. 구현 과제 (12월 11일 제출)
**과제명**: 공공데이터포털 활용 스마트시티 어플리케이션 구현

**요구사항**:
1. ✅ **RESTful API 제공** (필수, 미제공 시 감점)

   - **MongoDB 사용 시 가산점** ⭐
